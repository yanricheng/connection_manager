Openfire Connection Manager Module Readme

version: 3.6.3
released: 01/08/2009

Thank you for downloading the Openfire connection manager module!

Connection managers improve the scalability of the Openfire server by allowing multiple servers to handle the load of client connections. Each connection manager aggregates XMPP client connections and forwards their traffic to a central XMPP server. For more information and installation instructions, view the documentation that can be found in the "documentation" directory included with this distribution.

Further information can be found on the Openfire website.

Changelog

Connection manager module releases are synchronized with Openfire server releases. View the Openfire changelog for details on what's changed since the last release.
